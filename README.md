# Network Monitor

A Python-based network monitoring tool with a web dashboard.

## Features
- Pings devices and checks if they are online or offline
- Auto-refreshes every 30 seconds
- Web dashboard viewable in any browser
- Logs results to a file

## Technologies Used
- Python 3
- Flask
- Subprocess module

## How to Run

Install dependencies:
pip install flask

Run the terminal monitor:
python monitor.py

Run the web dashboard:
python dashboard.py

Then open your browser and go to:
http://localhost:5000
