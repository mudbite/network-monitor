# Network Monitor

A Python-based local network scanner with a web dashboard.

## Features
- Scans entire local network (192.168.1.0/24)
- Detects all devices connected to WiFi
- Displays results in a clean web dashboard
- Shows device IP and connection state

## Technologies Used
- Python 3
- Flask
- Nmap
- Python-nmap

## Requirements

Install dependencies:
pip install flask python-nmap

Install nmap:
pkg install nmap

## How to Run

python dashboard.py

Then open your browser and go to:
http://localhost:5000

## Note
Must be connected to WiFi for local network scanning to work.
